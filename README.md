# 🎮 扫雷游戏 - Web单页应用

## 项目简介
基于 HTML、CSS、JavaScript 开发的扫雷游戏单页应用。

## 功能特性
✅ 三种难度选择（初级、中级、高级）  
✅ 首发防雷保护  
✅ 双击快速清除  
✅ 右键插旗标记  
✅ 计时器和剩余雷数显示  
✅ 本地排行榜  
✅ 响应式设计

## 运行方式
1. 下载所有文件
2. 用浏览器打开 index.html

## 项目结构
