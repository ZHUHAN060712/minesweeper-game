\# 扫雷游戏 - Web单页应用



\## 项目简介

HTML+CSS+JavaScript开发的扫雷游戏。



\## 功能

1\. 三种难度选择

2\. 首发防雷保护

3\. 双击快速清除

4\. 右键插旗标记

5\. 计时器和雷数显示

6\. 本地排行榜

7\. 响应式设计



\## 运行

下载文件，用浏览器打开index.html
